Politique de Confidentialité

Application : Calculatrice Turbo | Mise à jour : 12 février 2026

Cette politique décrit comment l'application Calculatrice Turbo gère vos données. Nous nous engageons à limiter la collecte de données au strict nécessaire pour le fonctionnement de l'application et la diffusion de publicités permettant de maintenir la gratuité du service.

Pour diffuser des publicités (notamment les publicités récompensées pour débloquer le calculateur d’âge), nous intégrons le SDK [Google Mobile Ads (AdMob)](https://policies.google.com). Ce dernier collecte automatiquement :

  • Identifiants publicitaires : Advertising ID (Android) ou IDFA (iOS) pour la personnalisation des annonces.
  • Identifiants d'appareil : ID de l'ensemble d'applications (App set ID) pour lutter contre la fraude.
  • Localisation approximative : Déduite de votre adresse IP pour proposer des publicités pertinentes dans votre région/pays.
  • Données de performance : Logs de plantage et temps de réponse pour assurer la stabilité technique du SDK.

L’application fonctionne principalement en mode local :

  • Historique : Vos calculs et paramètres sont stockés dans une base de données SQLite **uniquement sur votre appareil**.
  • Confidentialité : Ces données ne quittent jamais votre téléphone et ne sont transmises à aucun serveur externe.
  • Contrôle : Vous pouvez vider cet historique à tout moment via les paramètres de l'application.

Certaines fonctionnalités avancées (ex: Calculateur d'âge) peuvent être débloquées temporairement après le visionnage d'une publicité. L'interaction avec ces publicités est traitée par Google. Vous pouvez désactiver la personnalisation des annonces dans les https://adssettings.google.com - Paramètres Google de votre téléphone.

Bien que nous ne collections pas de données personnelles directement, les données techniques traitées par AdMob sont transmises via le protocole sécurisé HTTPS. Le stockage local SQLite bénéficie des protections standards du système d'exploitation Android.

Conformément au RGPD et aux politiques de protection de la vie privée :

  • Droit de suppression : Désinstallez l'application pour supprimer instantanément toutes les données locales.
  • Réinitialisation publicitaire : Vous pouvez réinitialiser votre Advertising ID dans Paramètres > Google > Annonces.
  • Accès : Vos données de calcul sont visibles et modifiables directement dans l'interface de la calculatrice.
